"\r\n====\r\n".$m_comment;	$headers = 'From: fromm@hoge.co.jp' . "\r\n";	mb_send_mail($to, $subject, $body, $headers);	の http://hoge.co.jp、to@hoge.co.jp'、fromm@hoge.co.jp を書き換えてください4. コメントフォームを設置するHTMLへのコードの追加	(1) CSSをリンクする。通常<head></head>内へ記述		<link rel="stylesheet" href="/commentform/commentform.css" type="text/css">		(2)コメントフォームを設置する<body></body>の任意の箇所に記述		<div id="commentBox"></div>	<script type="text/javascript" src="/commentform/commentform.js"></script>	以上
