# 基本機能

キャラ図鑑の基本的な機能について説明します。

## 検索機能

### 名前検索

上部の検索バーでキャラクター名を検索できます。

#### 特徴
- **予測変換**: 入力中に候補が表示されます
- **部分一致**: 名前の一部でも検索可能
- **日英対応**: 日本語名・英語名の両方で検索可能
- **武器名検索**: 武器名でも検索できます（設定で有効化）

#### 使い方
1. 検索バーをクリックまたは`/`キーを押す
2. キャラクター名を入力
3. 予測候補から選択するか、Enterで検索
4. クリアボタン（×）で検索をリセット

#### ショートカット
- `/` - 検索フォーカス
- `↑`/`↓` - 候補の移動
- `Enter` - 検索実行/候補選択
- `Esc` - 候補を閉じる

### 検索オプション

#### 武器検索
設定メニューから「武器検索: 有効/無効」で切り替え可能。
有効にすると、武器名でもキャラクターを検索できます。

例: "エクスカリバー" → エクスカリバーを装備しているキャラクターを検索

#### カスタムタグ検索
設定メニューから「カスタムタグ検索: 有効/無効」で切り替え可能。
有効にすると、カスタムタグ名でも検索できます。

#### 予測変換の強調表示
設定メニューから「強調表示: 有効/無効」で切り替え可能。
有効にすると、予測候補の一致部分がハイライトされます。

## フィルター機能

### 基本フィルター

複数の条件でキャラクターを絞り込むことができます。

#### フィルター項目
1. **世界** - キャラクターの出身世界
2. **種族** - 人間、エルフ、獣人など
3. **戦闘スタイル** - 近接、遠距離、魔法、支援
4. **属性** - 火、水、風、土、光、闇など
5. **グループ** - 所属組織やグループ
6. **カスタムタグ** - ユーザーが作成したタグ

#### 使い方
1. フィルターボタンをクリック
2. 条件を選択（複数選択可能）
3. 「適用」をクリック
4. 「リセット」で全てのフィルターをクリア

### フィルター情報の表示

フィルター適用中は、検索バーの横に「i」アイコンが表示されます。

#### 使い方
1. iアイコンにマウスホバー（PC）またはタップ（モバイル）
2. 現在適用中のフィルター一覧が表示されます
3. ポップアップ内の「クリア」ボタンで全てのフィルターを解除

#### 表示内容
- 世界フィルター
- 種族フィルター
- 戦闘スタイルフィルター
- 属性フィルター
- グループフィルター
- カスタムタグフィルター

### AND/OR条件

同じ項目内では**OR条件**（いずれか）
異なる項目間では**AND条件**（全て）で絞り込まれます。

**例:**
- 種族: 人間 OR エルフ
- 戦闘スタイル: 近接
- → 「人間または、エルフ」かつ「近接」のキャラクター

## 表示機能

### カード表示

キャラクターはカード形式で表示されます。

#### カード情報
- キャラクター画像
- 名前（日本語/英語）
- 武器アイコン（設定で表示/非表示）
- レアリティ表示

#### 武器アイコン表示設定

設定メニューから3つのモードを選択できます:
1. **常に表示** - 全てのカードに武器アイコンを表示
2. **フィルター時のみ** - フィルター適用時のみ表示（デフォルト）
3. **非表示** - 武器アイコンを表示しない

### 詳細表示

カードをクリックすると、詳細情報がポップアップ表示されます。

#### 詳細情報の内容
- **基本情報**
  - 名前（日日本語/英語）
  - 世界
  - 種族
  - 戦闘スタイル
  - 属性
  - グループ

- **武器情報**
  - 武器名
  - 武器種別
  - 武器アイコン

- **関連キャラクター**
  - 関係のあるキャラクター一覧
  - クリックで該当キャラクターの詳細を表示

- **関係関連**
  - 家族、友人、敵対など
  - 関係の説明文

- **カスタムタグ**
  - 付与されているタグ一覧
  - タグの追加/削除

#### 詳細画面の操作
- **×ボタン** - ポップアップを閉じる
- **Escキー** - ポップアップを閉じる
- **関連キャラクタークリック** - そのキャラクターの詳細を表示
- **タグ追加** - カスタムタグを追加
- **タグ削除** - タグの×ボタンで削除

### レスポンシブ表示

画面サイズに応じて自動的にレイアウトが変更されます。

#### PC表示（1024px以上）
- カード: 4列グリッド
- 詳細画面: 横幅広めのポップアップ
- ホバー効果あり

#### モバイル表示（1024px未満）
- カード: 2列グリッド
- 詳細画面: 全画面表示
- タッチ操作最適化

## テーマ

3つのテーマから選択できます。

### ライトテーマ（デフォルト）
- 明るい背景
- 読みやすい黒文字
- 日中の使用に最適

### ダークテーマ
- 暗い背景
- 白文字
- 夜間の使用に最適
- 目に優しい

### モダンテーマ
- グラデーション背景
- カラフルなUI
- スタイリッシュなデザイン

### テーマ切り替え
1. ハンバーガーメニューを開く
2. 「テーマ切替」をクリック
3. 3つのテーマが順番に切り替わります

## 言語

日本語と英語の2言語に対応しています。

### 切り替え可能な項目
- キャラクター名
- 世界名
- 種族名
- 戦闘スタイル名
- 属性名
- グループ名
- UI要素

### 言語切り替え
1. ハンバーガーメニューを開く
2. 「言語切替」をクリック
3. 日本語/英語が切り替わります

### 検索への影響
両言語の名前で検索が可能です。
例: "アリス"でも"Alice"でも検索できます。

- Firefox 最新版
