## クイックスタート

### 基本的な使い方

1. **キャラクター検索**: 上部の検索バーに名前を入力
2. **フィルター**: 世界、種族、戦闘スタイル、属性などで絞り込み
3. **詳細表示**: カードをクリックして詳細情報を表示

### 編集モード

1. ハンバーガーメニュー（☰）を開く
2. 「編集モード: オン」をクリック
3. サブモードを選択:
   - **選択**: キャラクターを選択して一括タグ追加
   - **並び替え**: ドラッグ&ドロップでカード順序を変更

### カスタムタグ

1. ハンバーガーメニューから「カスタムタグ」を開く
2. 「新しいタグを作成」をクリック
3. タグ名と色を設定
4. キャラクターにタグを追加

## 主な機能一覧

### 検索・フィルター機能
- 名前検索（予測変換付き）
- 武器名検索
- 複数条件フィルター（世界、種族、戦闘スタイル、属性、グループ）
- カスタムタグフィルター
- フィルター情報の表示

### 表示機能
- ライト/ダーク/モダンテーマ
- 日本語/英語切り替え
- 武器アイコン表示設定
- レスポンシブデザイン（PC/モバイル対応）

### 編集機能
- キャラクター選択モード
- 一括タグ追加
- ドラッグ&ドロップ並び替え
- カスタム並び順の保存

### データ管理
- キャラクターデータのエクスポート
- カスタムタグのエクスポート
- JSONデータのインポート
- IndexedDBによるローカル保存

### 並び替え機能
- ID順
- 名前順（50音順/アルファベット順）
- ランダム順
- カスタム順（ドラッグ&ドロップ）
- タグごとの個別並び順設定

## ショートカットキー

- `/` - 検索フォーカス
- `Esc` - ポップアップを閉じる
- `↑`/`↓` - 検索予測候補の移動
- `Enter` - 検索実行/候補選択

## ブラウザ対応

- Chrome/Edge (推奨)
- Firefox
- Safari
- モバイルブラウザ

## データの保存先

- **キャラクターデータ**: IndexedDB (`CharactersDB`)
- **カスタムタグ**: IndexedDB (`CustomTagsDB`)
- **設定**: localStorage
- **並び順**: localStorage

## トラブルシューティング

### データが表示されない
1. ブラウザのコンソールでエラーを確認
2. IndexedDBが有効か確認
3. キャッシュをクリアして再読み込み

### 並び替えが保存されない
1. localStorageが有効か確認
2. プライベートモードでないか確認

### カスタムタグが消える
1. IndexedDBのデータを確認
2. データ管理からエクスポートしてバックアップ
